## What It Improves

### Clarity and Cleanliness
Streaming platforms, lossy codecs, and repeated compression passes leave invisible damage — spectral wobble, phase smearing, and high-frequency grit that exposes itself on a club system. Stellare identifies and removes it before any enhancement is applied.

- **Phase and Spectral Dewobble** — Flutter and phase drift corrected in a dual-pass approach, once on the raw source and again after excitation
- **Artifact Removal** — Identifies and attenuates compression ghosts, pre-ringing, and codec damage without touching transient content
- **Source Normalisation** — Phase coherence enforced across the stereo field before processing begins

### High Frequency and Air
Streaming encodes remove the 14–18 kHz air from electronic music. Stellare restores it using controlled harmonic generation, not synthetic noise.

- **Harmonic Exciter** — Targets 3200 Hz for club formats and 2350 Hz for streaming, filling the spectral ceiling that compression empties
- **Full Spectral Coverage** — Coverage increased from 78.8% to 100% on a single pass — rated EXCELLENT
- **Controlled Output** — Frequency-aware limiting on the exciter path prevents added air from becoming harsh or fatiguing

### Tonality and Genre Intelligence
- **Genre-Aware EQ** — Detects Deep House, Techno, Tech House, Trance, Melodic House and more
- **Dynamic EQ Response** — Boosts and cuts respond to the track's own crest factor and dynamic range
- **BPM Detection** — Tempo detected and embedded in metadata automatically

### Loudness and Limiting
Stellare uses HPSS (Harmonic-Percussive Source Separation) to process kick and pad on independent paths before limiting — a technique used in professional mastering chains.

- **Streaming LUFS Targeting** — Output calibrated to −13 LUFS, the Spotify and Apple Music standard
- **8× Oversampled ARC Limiting** — Catches inter-sample peaks that standard limiters miss
- **HPSS Split Processing** — Kick and pad processed independently so neither crushes the other

### Distribution Ready
- **Auto Metadata and Genre Tagging** — ID3 tags written automatically — genre, BPM, and release data
- **Beatport and Streaming Filename Format** — Output files renamed to Stellare release standard
- **Automatic Archive Cleanup** — All intermediate processing files removed after a successful run

## The 10-Stage Pipeline

| Stage | Process | Description |
|---|---|---|
| 0 | Format Converter | Converts MP3, FLAC, AAC, MP4 to 44.1 kHz 32-bit WAV |
| 2 | Spectral Dewobble I | Phase correction and spectral flutter removal — first pass |
| 3 | Artifact Removal | AI-powered detection and removal of compression artefacts |
| 5.5 | Source Processor | Phase coherence enforcement and stereo field normalisation |
| 5 | Harmonic Exciter | High-frequency extension via harmonic generation |
| 6.5 | Spectral Dewobble II | Second dewobble pass post-excitation |
| 6 | Auto EQ | Genre-aware dynamic EQ with BPM detection |
| 7 | Limiter & LUFS | 8× oversampled ARC limiting — targets −13 LUFS |
| 8 | MetaTag | Writes ID3 tags, genre, BPM, renames file |
| 10 | Cleanup | Removes intermediate files, leaves final master |
